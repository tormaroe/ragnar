
# Project summary

- We are creating a programming language called Ragnar, hosted in .NET/C#. 
- The syntax and semantics are based on the Rebol language. 
- Features include blocks, functions, loops, recursion, and .NET interop.
- Interop with .NET is an important feature.
- Ragnar can read and execute source files and provides an interactive REPL.
- We will work in small steps, and write unit tests as we go.
